# crispr-nf

Pre-processing of CRISPR-Cas9 / shRNA screening data

## Basic usage

With Singularity >= 2.3.x
```bash
nextflow run ZuberLab/crispr-nf
```

Local execution
```bash
nextflow run ZuberLab/crispr-nf -profile local
```

## Documentation
```bash
nextflow run ZuberLab/crispr-nf --help
```
